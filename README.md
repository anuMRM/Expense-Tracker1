#  Expense Tracker Pro

Developed a personal finance management web application using Python and Streamlit with secure user authentication, expense/income tracking, SQLite database integration, and interactive visualization dashboards.

## 🚀 Features

- User authentication
- Secure signup and login
- Add income records
- Add and categorize expenses
- Expense history tracking
- Search and filtering
- Delete expenses
- Interactive dashboard
- Expense category visualization
- Spending trend analysis
- SQLite database integration

## 🛠 Technologies Used

- Python
- Streamlit
- SQLite
- Pandas
- Plotly
- bcrypt

## 📊 Dashboard

The application provides:

- Total income calculation
- Total expense calculation
- Balance tracking
- Category-wise expense charts
- Spending trends
